# Chat-SpringCloud
### 项目介绍 

基于Spring Cloud实现的基于Web的为微服务聊天系统。

本项目由 [FreudFan/chat](https://github.com/FreudFan/chat) 项目重构升级而来。

### 项目源码

| github           |                                                              |
| ---------------- | ------------------------------------------------------------ |
| Spring Cloud版本 | https://github.com/FreudFan/chat-springcloud |
| Spring Boot版本  | https://github.com/FreudFan/chat |

### 模块介绍

| 功能           | 应用名      |
| :------------- | :---------- |
| 注册中心       | eureka      |
| API网关        | api-gateway |
| 用户服务       | user        |
| 群组服务       | group       |
| socket通讯服务 | message     |
| 文件服务       | file        |
